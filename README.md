# 前端机器学习———光伏功率预测系统

## 基本介绍
基于浏览器制作一个简单的AI,用户只需通过简单的鼠标点击来预测一段时间后的光伏发电功率，从而为电网负载均衡提供帮助。

## 在线浏览地址
http://47.254.216.86:8080/#/

## 技术栈
* 机器学习库 : TensorflowJS
* 框架: vue
* 图表: echart
* 界面: elementUI
